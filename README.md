# management

> blog management project

## stack

vue + element + axios

## Build Setup

+ run server

[api](https://github.com/Cacivy/koa-typescript)

+ local

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
